# Spring Boot AWS Deployment Project

A cloud-based Spring Boot application deployed on AWS using AWS RDS and AWS Elastic Beanstalk.

## Features
- REST API Development using Spring Boot
- MySQL Database Integration
- AWS RDS Database Configuration
- AWS Elastic Beanstalk Deployment
- Maven JAR Packaging
- Cloud Hosted API Testing

## Technologies Used
- Java
- Spring Boot
- MySQL
- AWS RDS
- AWS Elastic Beanstalk
- Maven
- REST API

## Project Highlights
- Developed REST APIs using Spring Boot
- Connected application with AWS RDS MySQL Database
- Generated executable JAR file using Maven
- Successfully deployed application on AWS Elastic Beanstalk
- Tested cloud-hosted APIs successfully
